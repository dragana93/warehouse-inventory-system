# Warehouse Inventory System

## Purpose

Application for managing warehouse inventory.

## Functional Requirements

### Products

- Add product
- Edit product
- Delete product
- View product details

### Inventory

- Increase stock
- Decrease stock
- Stock cannot be negative

### Categories

- Create category
- List categories

### Filtering

- Filter by:
  - Name
  - Category
  - Code

### Sorting

- Name
- Price
- Quantity

### Reports

- Total stock per category

### Audit Log

Record every inventory change

## Non Functional Requirements

- 70%+ unit test coverage
- Error logging
- Global error handling
- SQLite database
